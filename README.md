# Features
* Supports Russian and English languages (bot communication, search requests and information about heritage sites)
* The most complete database of site locations (most sites contain more than one location listed on [UNESCO website](https://whc.unesco.org/en/list/)
* Searches nearest sites by the closest location
* Text search by name, country, inscription year, etc.
* Search by several database fields using natural language logic (and/or)
* Automatic database updates from [UNESCO xml file](https://whc.unesco.org/en/list/xml)

# Stack
* Deployed to Heroku
* Runs on Node.js using Telegram webhooks
* Telegram API using [Telegraf/Micro-bot](https://github.com/telegraf/micro-bot) framework
* Database stored remotely with MongoDB
